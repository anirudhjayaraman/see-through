# See Through the Hype 🔍✨

A premium Chrome Extension designed to help you separate real substance from marketing hype. Using Gemini AI, it analyzes LinkedIn posts, X threads, and landing pages to identify jargon, hidden assumptions, and missing evidence.

## Features
- **Modern Minimal UI**: A clean, glassmorphism-based side panel.
- **Deep Extraction**: Specialized selectors for LinkedIn and X to ensure high-quality text capture.
- **Credibility Scoring**: 0-100 transparency score based on claim specificity and evidence.
- **Privacy First**: Minimal permissions, using `activeTab` to only access the page when you click.
- **Demo Mode**: Built-in test cases for fool-proof presentations.

## Installation
1.  **Clone** this repository.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** (top right).
4.  Click **Load unpacked** and select the root folder of this project.

## Setup
1.  Open the **Side Panel** by clicking the eye icon in your browser toolbar.
2.  Go to **Settings** (gear icon).
3.  Enter your **Gemini API Key** (from [Google AI Studio](https://aistudio.google.com/app/apikey)).
4.  *Optional*: Enter `DEMO` in the API Key field to try the extension without a real key using the included mock samples.
